元数据

网站的元数据定义在 <head> 标签中。元数据不会显示在网页中，但会被浏览器解析。

常用的有设置网页字符集。

标签

HTML 的标签都有其语义，应在合适的地方使用合适的标签。

标题标签

标题标签用于显示文档标题，总共有六级标题。

段落标签

段落标签 <p> 用于定义一个段落，它会根据浏览器窗口大小自动换行，段落与段落之间会留有一定空隙。

换行标签

<br /> 标签可以强制换行。

文本格式化标签

盒子标签

<div> 和 <span> 标签是盒子标签，没有语义，只用于盛放内容。

<div> 是块级元素，独占一行。

<span> 是内联元素，不会换行。

图像标签

<img> 标签可以在网页中插入图片。

src 属性表示图片的路径，可以使用本地文件的绝对路径或相对路径，网页链接。

alt 属性可以在图片不能显示时展示提示信息。

title 属性可以在鼠标悬停在图片时显示提示信息。

超链接标签

<a> 标签可以为网页插入超链接。

href 属性指定超链接的目标。可以是本地文件或网页连接。# 表示空链接。

target 属性指定在何处打开超链接，_blank 表示新标签页打开，_self 表示在当前标签页打开。默认当前标签页打开。

通过与 id 属性配合，可以实现页面内跳转。

注释标签

特殊字符

一些字符，如空格，左右尖括号等，无法在 HTML 文档中显示，需要用一些其他符号替代。

  • &nbsp; 空格
  • &lt; 小于号
  • &gt; 大于号
  • &amp; 和号

表格标签

基本表格

  • <table> 表示一个表格
  • <th> 表示表头单元格（只有一格）
  • <tr> 表示表中一行
  • <td> 表示一行中的一格

表格结构标签

为了清晰划分表格结构，引入表格结构标签 <thead> 和 <tbody>。

合并单元格

  • rowspan 跨行合并（竖向）
  • colspan 跨列合并（横向）

列表

无序列表

有序列表

自定义列表

自定义列表没有序号或者格式。

表单

表单域是一个包含表单的区域，通过 <form> 定义。

输入元素

type 的值确定 <input> 标签的类型。

  • text 普通文本框
  • password 密码输入框
  • radio 单选按钮
  • checkbox 复选框
  • submit 提交按钮

单选框和复选框可以设置 checked 属性，表示默认选中。

submit 类型可以在点击后将表单数据提交到 action 指定的文件。

button 类型会创建一个按钮，通常结合 JS 使用。

<label> 标签

<label> 通常与一个表单元素绑定。

当点击用户名 时，浏览器会自动聚焦到输入框。

下拉元素

<select> 标签表示一个下拉菜单。

至少包含一对 <option> 标签。<option> 可加 selected 属性，表示默认选中。
