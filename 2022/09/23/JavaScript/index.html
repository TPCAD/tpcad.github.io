JavaScript

JavaScript 的书写位置

行内式

JS 代码可以直接写在 HTML 的标签中。

内嵌式

JS 代码可以写在 script 标签内。

外部 JS

JS 也可以像 CSS 一样保存在外部文件中。需要放在 body 标签中，且在最下方，以保证其他 HTML 代码都已加载完毕。

注释

简单输出

console.log()

console.log() 可以将信息输出到控制台。

alert()

alert() 可以在网页中弹出一个警示框。

变量

数据类型

JS 是弱类型语言，变量不需要声明类型。

typeof 操作符可以用来检测操作数的数据类型。

String

JS 用单引号或双引号定义字符串。

Number

JS 只有一种数据类型。数字可以带或不带小数点。支持科学计数法。

数字型支持不同的进制表示。conlose.log() 会将数字自动转换成十进制输出。

数字型最大与最小值

数字型有三个特殊值

  • Infinity：正无穷
  • -Infinity：负无穷
  • NaN：Not a Number

isNaN() 可以判断值是否为 NaN。

Boolean

布尔类型只有两个值，true、false。

Undefined

当变量未被定义时，它就是一个 Undefined。

Null

null 表示空值。可以用 null 来清空一个变量。

for

函数

字符串

字符串是一种基本类型，不具备属性和方法。但也可以用 new 创建字符串（不建议这样做）。基本类型的字符串也可以使用属性和方法，因为 JavaScript 在执行方法和属性时可以把它当作对象。

转义字符

转义字符 含义
' ‘
" “
\ \
\n 换行
\r 回车
\t tab(制表符)
\b 退格符
\f 换页符

字符串长度

字符串拼接

+ 可以将字符串与其他数据类型进行拼接。

常用的字符串方法

charAt()

返回指定索引为止的字符（以字符串类型返回）。

concat()

连接两个或多个字符串，返回连接后的字符串。

indexOf() 和 lastIndexOf()

返回字符串中检索指定字符第一次/最后一次出现的位置

toLowerCase() 和 toUpperCase()

把字符串转换成小写/大写。

trim()

去除字符串前后的空白。

substr()

从起始索引号提取字符串中指定数目的字符。

substring()

提取字符串中两个指定的索引号之间的字符（左闭右开区间）。
