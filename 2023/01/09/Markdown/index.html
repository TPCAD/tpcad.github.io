Markdown

标题

使用 # 标记 1～6级标题。# 后要有一个空格。

一级标题是一份文档中最大的标题，一份文档中应只有一个一级标题。

段落

可以在段落末尾加上两个空格表示段落换行。

也可以在段落后使用一个空行来表示段落换行。

文本样式

并不是所有的 markdown 解析器都支持所有的语法格式。

粗体斜体删除线下标上标下划线

列表

无序列表

可以用 *、+、- 来标记无序列表。符号后要有一个空格。

  • 第一项
  • 第二项
  • 第三项

有序列表

数字加上 . 标记有序列表。. 后要有一个空格。

  1. 第一项
  2. 第二项
  3. 第三项

嵌套列表

  1. 第一项
    • 嵌套第一项
      1. 嵌套嵌套第一项上标

引用

用 > 来标记一个引用。> 后要有一个空格。

这是一个引用。

嵌套引用。

还可以在引用中嵌套列表或在列表中嵌套引用。

代码块

行内代码块

用 ` 包裹行内代码块。

printf("\n");

如果想高亮显示 `，可以使用双倒引号，且双倒引号与文本内容之间必须有空格。

代码块

使用 ``` 包裹代码块。还可以选择对应语言的高亮显示。

使用四重倒引号包裹三重倒引号即可显示三重倒引号。

超链接

这是一个超链接。

图片

表格

使用 | 分隔单元格，使用 - 分隔表头与其他行。

对齐方式：

  1. :- 该列左对齐
  2. -: 该列右对齐
  3. :-: 该列居中对齐

未设置对齐的表格：

表头行 表头行
单元格不对齐 单元格不对齐
单元格 单元格

设置对齐的表格：

表头行 表头行 表头行
单元格左对齐 单元格居中对齐 单元格右对齐
单元格 单元格 单元格

任务列表

  • 空格表示未完成
  • x表示已完成

HTML 支持

Markdown 支持使用 HTML 标签来进行文字段落的书写。常见的粗体、斜体以及超链接，图片的插入都可以通过 HTML 来实现。
